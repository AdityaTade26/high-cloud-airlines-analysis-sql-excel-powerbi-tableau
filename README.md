## Business Problem
Airline leadership struggled with limited visibility into load factor performance,
passenger demand trends, and route efficiency. Although data was available,
actionable insights were missing, leading to inefficient routes and revenue loss.

## Objective
To transform raw airline operations data into actionable insights using
SQL, Excel, Power BI, and Tableau.

## Tools & Technologies
- SQL: Data analysis, aggregations, time-based trends
- Excel: Data cleaning, KPI preparation, exploratory analysis
- Power BI: Interactive executive dashboard with slicers and drill-downs
- Tableau: Visual storytelling and performance comparison dashboards

## Key KPIs Analyzed
- Load Factor
- Total Passengers
- Seats Utilization
- Route & Carrier Performance
- Weekday vs Weekend Trends
- Distance-based Flight Analysis

## Key Insights
- Load factor peaks in Q3 across multiple years
- Weekend flights consistently outperform weekdays
- High-performing routes and carriers identified for optimization
- Insights suggest opportunities for dynamic pricing strategies

## Business Impact
This analysis enables airline stakeholders to quickly identify
high-performing routes, optimize seat utilization, and improve
revenue-driven decision-making.
